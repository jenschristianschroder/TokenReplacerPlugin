# TokenReplacerPlugin
Dataverse Plugin to replace Token with Value

# Plugin Unsecure Configuration
```
{ 
   "Token": "[newlinetoken]",
   "Value": "\n",
   "Fields": [ 
      "prefix_fieldname"
   ] 
}
```
